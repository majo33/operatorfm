Select/deselect files

click icon to select or deselect a file
longclick icon to select or deselect all

Add/remove tab

press New tab icon to create a new tab
longpress New tab icon to remove tab

Context menu

longpress any item to display menu
depends on selection & clicked item

Bookmarks

longpress any item & choose Bookmark...
longpress the bookmark icon to delete

Copy files

select files on first tab
swipe to destination tab
longpress any item
choose Copy action

Move into

select files, don't change tab
longpress destination directory
choose Move into action

Delete a file

no active selection
longpress the item
choose Delete action

Delete multiple files

select files/directories
longpress any item
choose Delete action
